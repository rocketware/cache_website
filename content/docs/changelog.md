# Changelog

All notable changes to Cache.

---

## Current Status

**Version 0.8 ALPHA** — Core functionality is stable and in daily use. Working toward 1.0 release.

---

## [1.0.0] - Coming Soon

Initial public release.

### Core Features

- **Note-taking engine**: Custom markdown editor with real-time syntax highlighting
- **Git sync**: Clone, pull, commit, push—full repository management without external dependencies
- **Modal editing**: Vim, Helix, and Simple (non-modal) configurations
- **Timed notes**: Date-based journaling with calendar navigation
- **Reference notes**: Folder-based organization for non-temporal content
- **Full-text search**: SQLite FTS5 with line-level results and context
- **Knowledge graph**: Automatic link tracking with PageRank-style importance scoring
- **Theme support**: TOML-based theming with dark and light modes

### Platforms

- macOS (App Store and direct download)
- Linux
- iOS
- Android (Google Play and de-googled builds)

### Privacy

- No analytics, no tracking, no third-party SDKs
- Accountless licensing via PGP-signed tokens
- Your data stays in your Git repository
