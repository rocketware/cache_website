Cache v0.5.1

May 01, 2026

← Releases

Changelog

  • Add iOS support
  • Add auto-commit on idle — automatically commits the current note after a configurable period of inactivity
  • Add auto-commit for config changes — settings and directory config edits are now committed automatically
  • Add task reminders — configurable on-start and periodic notifications summarising due, overdue, and upcoming tasks
  • Add uncommitted changes reminder — periodic notification when files haven't been committed
  • Add on-start options for auto-pull and uncommitted reminder — run once per session or at a configurable interval
  • Add license sync — refresh or open the license file directly from license settings
  • Add automatic detection and recovery when .git folder is missing
  • Add swipe down to dismiss keyboard
  • Improve navigation speed — auto-commit and sync now run in the background
  • Improve text input fields — context-aware keyboard types for URLs and emails, autocorrect disabled for credentials and identifiers
  • Improve sync setup prompt — dismissing now remembers the choice for the rest of the day
  • Change macOS app icon
  • Fix issue when switching note repositories affecting some devices
