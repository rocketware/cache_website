Getting Started

Cache is a personal knowledge management app that stores your notes as plain markdown files synced via Git. You own your data completely — no accounts, no cloud lock-in, no data collection.

Initial Setup

On first launch, choose to clone an existing repository or start fresh. If you create a new repository, you can customise the initial note types and optionally set up a Git remote for backup and sync.

Tip: When setting up a new repository, include the Tutorial note type. This adds getting-started guides directly in your repository. Tutorial notes aren't synced via Git, so you can delete them once you're comfortable.
